Printer Setup & Installation

Epson printer setup and installation offers pre-configured, on-site printer services that makes switch easy.

Setup and Install printer

Setting up your printer with two steps: First Connect your printer to computer then Install Printer Driver.

Steps to Setup Epson Printer in Computer & Mac

  • Turn on the printer.
  • Let the system detect and install the printer.
  • Install printer software from disc or download.
  • Run the driver file to install.
  • Follow on-screen instructions.
  • Printer is ready to use.

Steps to Setup Wireless Epson Printer

Follow easy steps to easily setup the Epson printer to Wi-Fi network:

  • Turn on the wireless printer.
  • Connect to Wi-Fi network using printer's control panel.
  • Install printer drivers on computer.
  • Select wireless printer during installation.
  • Complete setup; printer ready.

Steps to Setup Wired Epson Printer

Follow easy steps to easily setup the Epson printer to wired connection:

  • Connect the printer to the router.
  • Turn on the printer and press the LAN button.
  • Select the ‘LAN Setup’.
  • Follow the on-screen instructions to complete the setup process.
